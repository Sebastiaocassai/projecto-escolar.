Introdução à Linguagem CSS

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). Ele é usado para estilizar páginas HTML, ou seja, deixar o site bonito e organizado.

  • Mudar cores
  • Definir tamanhos de texto
  • Ajustar fontes
  • Criar layouts (menus, colunas, rodapés)
  • Controlar espaçamento e posicionamento

O HTML cria a estrutura do site. O CSS cria e cuida da aparência.

Importância do CSS no Desenvolvimento Web

  • Separa conteúdo da aparência
  • Deixa o site mais bonito e profissional
  • Facilita a manutenção
  • Permite criar sites responsivos (celular, tablet e computador)
  • Melhora a experiência do usuário

Tipos de Folhas de Estilo (CSS)

  1. CSS Inline
  2. CSS Interno
  3. CSS Externo

O que são Seletores em CSS?

Seletores são usados para escolher quais elementos HTML serão modificados pelo CSS. 

p {
    color: blue;
}

Tipos de Seletores CSS

  1. Seletor de tipo (tag)
  2. Seletor de classe
  3. Seletor de ID
  4. Seletor universal
  5. Seletor de agrupamento
  6. Seletor descendente
  7. Seletor filho direto
  8. Seletor de atributo
  9. Pseudo-classe
  10. Pseudo-elemento
