O que são seletores em CSS? Os seletores são usados para "selecionar" elementos HTML que você quer modificar com regras de estilo. Exemplo básico: p { color: blue;} Aqui, o seletor p seleciona todas as tags p da página e define a cor do texto como azul

Tipos de Seletores CSS

1º seletor de tipo(ou de tag)

2º seletor de classe (classe)

3º seletor de ID (ID)

4º seletor universal

5º seletor de agrupamento

6ºseletor descendente

7º seletor filho direito

8º seletor de atributo

9º seletor pseudo-classe

10º seletor pseudo-elemento
