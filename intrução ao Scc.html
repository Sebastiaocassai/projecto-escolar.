Introduçao A linguagem Css

Css siginifica cascading style sheets (folhas de estilo em cascata, ele e usado para estelizar paginas htlm, ou seja deixa o site bonito e organizado. com o css voçe pode. mudar cores, definir tamanhos de texto, ajusta fontes. criar layouts (menus, colunas, rodapes) controlar espaçamento e posicionamento

O HTML criar a estrutura do site O CSS criar e cuida da aparençia

Importançcia do css no desenvolvimento web

O Css e muito importante porque separa conteudo de apareçia o html fica limpo e organizado e o css cuida do visual. deixa o site mais bonito e profissional permite criar designs e modernos e agradaveis

facilita a manutençao mudando um unico arquivo css, varias paginas podem ser alteradas ajuda na adaptaçao para celulares , com css e possivel criar sites responsivos (que funcionam bem no celular tablet e computadores

melhora a experençcia do usuario um site bem estelizado e mais facil e agradavel de usar

tipos de folhas de estilos (scc) 1.Scc inline. 2.Css interno (ou incorporada) 3.Css externo
